Box sizing

The div below contains two boxes that have the same width and height. The red box has box-sizing: content-box (the default value), while the blue box has box-sizing: border-box.

The red box's width and height describe only the content, and does not include the padding. The blue boxes width and height is of the whole box (including padding and border).

Summary

CSS property Value Effect
box-sizing content-box Width and height limit the content array of the element
box-sizing border-box Width and height describe the whole element, including padding and border.
